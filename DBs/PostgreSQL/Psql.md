### Install

```bash
$ sudo apt install -y postgresql-client 
```

### Connect

```bash
psql postgresql://$PSQL_USER:$PSQL_PASSWORD@$PSQL_HOST:$PSQL_PORT/$PSQL_DBNAME

# example
psql postgres://visiuspgadmin@visius-az-stage:xJHTcUi4PWDMqFF7HUaY@visius-az-stage.postgres.database.azure.com:5432/postgres
```

### List tables

```bash
db=> \dt
```
