### Help

```bash
man pct
```

### Enter CT shell

```bash
# no password needed
pct enter 100
```

### List CTs

```bash
pst list
```

### Config

```bash
pst config 100

# set auto boot
pst set 100 -onboot 1

# change ram
pst set 100 -memory 1024
```

### Start/Stop/Restart

```bash
pct start 100

pct shutdown 100

pct restart 100
```
