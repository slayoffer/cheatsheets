### Install

```bash
# CentOS
curl -sL https://rpm.nodesource.com/setup_13.x | sudo bash -
sudo yum install -y nodejs
# same as (add and enable Node repo on server)
curl --silent --location https://rpm.nodesource.com/setup_13.x | sudo bash -
```

### Check version

```bash
node -v
```

### Run .js

```bash
node app.js
```
