### Open in view mode

```bash
nano -v text.txt # -v for view mode
```

### Open on line number

```bash
nano +15 text.txt # open with cursor on line 15
```

### Save file

```bash
CTRL + o
```

### Exit file

```bash
CTRL + x
```

### Minimize Nano

```bash
CTRL + t + z
```

### Search text

```bash
CTRL + w
```
