### Basics

```bash
# Утилита netstat отображает различную статистическую информацию: сетевые подключения, статистику интерфейсов, таблицы маршрутизации, состояние портов и т.п.).

netstat -nap 

# example
sudo netstat -nap | grep ssh | grep -v tcp6
# -p — показывает имя приложения
# -a — показывает состояние сокетов, без него не узнать, какой порт использует приложение
# -n — показывает ip адреса, а не сетевые имена
# Без sudo нельзя получить информацию о запущенных приложениях — прав пользователя не хватит
```

### Show ports
