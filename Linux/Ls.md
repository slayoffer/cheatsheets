### Best LS tool

```bash
exa
```

### Long list

```bash
ls -l
```

### Human readable

```bash
ls -h
```

### List hidden

```bash
ls -a
```

### Show file types

```bash
ls -F
```

### Sort by mod time

```bash
ls -lhtr
# or other way round
ls -lt
```

### With modified time format

```bash
ls -lhF --time-style=long-iso --color=auto --ignore=lost+found
```

### Show Inodes

```bash
ls -i
```
